Política de Privacidade – Anti-Spam de Chamadas

Última atualização:

Este aplicativo (“Anti-Spam de Chamadas”) foi desenvolvido para ajudar o usuário a bloquear chamadas indesejadas de números que não estão salvos na agenda telefônica.

A sua privacidade é importante. Esta Política de Privacidade explica de forma clara como as informações são tratadas pelo aplicativo.

1. Informações que o aplicativo acessa

O aplicativo solicita apenas as permissões necessárias para seu funcionamento.

Acesso aos contatos (READ_CONTACTS)

Essa permissão é utilizada exclusivamente para:

  • Verificar se o número de uma chamada recebida está salvo na agenda do usuário
  • Decidir se a chamada deve ser permitida ou bloqueada automaticamente

Os contatos não são copiados, armazenados ou compartilhados. A verificação ocorre apenas no momento da chamada.

2. Informações armazenadas localmente

O aplicativo mantém uma lista local de números bloqueados, armazenada somente no dispositivo do usuário, com as seguintes finalidades:

  • Exibir ao usuário quais chamadas foram bloqueadas
  • Fornecer histórico e transparência do funcionamento do aplicativo

Essa lista:

  • Contém apenas números de telefone
  • Não é enviada para servidores externos
  • Não é compartilhada com terceiros
  • É removida automaticamente ao desinstalar o aplicativo

3. Informações que o aplicativo não coleta

O aplicativo não coleta, não transmite e não compartilha:

  • Nomes de contatos
  • Conteúdo da agenda
  • Gravações de chamadas
  • Localização
  • Dados de conta
  • Dados de navegação
  • Informações pessoais identificáveis
  • Informações do dispositivo

4. Uso e segurança dos dados

  • Todo o processamento ocorre localmente no dispositivo
  • Nenhuma informação é enviada para a internet
  • O aplicativo não utiliza servidores externos
  • O aplicativo não exibe anúncios

5. Funcionamento do aplicativo

  • Chamadas de números salvos na agenda são permitidas normalmente
  • Chamadas de números não salvos podem ser bloqueadas automaticamente
  • Os números bloqueados podem ser exibidos em uma lista local
  • O usuário pode ativar ou desativar o bloqueio a qualquer momento

6. Compartilhamento de dados

O aplicativo não compartilha dados com empresas, anunciantes, serviços de terceiros ou órgãos governamentais.

7. Privacidade infantil

Este aplicativo não é direcionado a crianças e não coleta intencionalmente dados de menores de 13 anos.

8. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada ocasionalmente. Recomenda-se que o usuário revise esta página periodicamente para se manter informado sobre como suas informações são tratadas.
